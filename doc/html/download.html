CCSL source and Documentation Files

CCSL source files can be downloaded from the eps_campus repository. Files available are:
0README describes the content and use of ccsl_source.tar
ccsl_4.4.xx.tar.gz Gzipped archive containing the complete CCSL MK4.4.xx source code, and test data with associated perl scripts
libmk4.f CCSL MK4.4 Fortran main library source code as used at ILL(gfortran version)
maimk4.f CCSL MK4.4 Fortran source code of Main programs used at ILL (gfortran version)
pigmk4.f.gz Fortran source code of graphics drivers used at ILL (gfortran version)
Mag3dgl.dmg Disc Image of MAC OSX program to display magnetic structures

PDF versions of the CCSL manual and Appendices are available in the CCSL/doc directory
CCSLman.pdf CCSL MK4.4 users manual in PDF format
appendixA.pdf Appendix A to the CCSL MK4.4 users manual in PDF format
appendixC.pdf Appendix C to the CCSL MK4.4 users manual in PDF format
appendixD.pdf Appendix D to the CCSL MK4.4 users manual in PDF format
appendixE.pdf Appendix E to the CCSL MK4.4 users manual in PDF format
Some Test Data are also available:
CCSLtests Perl script to run tests on the CCSL programs
testdata A directory containing test data (with expected results) for CCSL programs
